This program assumes that autoMicromanager Console has been started.  You will get a basic setup with a console output in the background that will log all actions in the system.

Start by clicking the Create config File

 

 

You must select the hardware that you wish to control.  The libariry is listed first and then the individual adapters.  Hilight the desired adapter and then use the select button to move it over.  Click Next Step when done.

 

You must now name the different controls that will be available to the system.  This is required for scripting to work as well as to allow the different components in the system to sync up with each other.  Give each thing a generic name.

Now you must define how you wish to interact with each of the device adapters.  A property list is the most minimal method, but for most of the other adapters, there is a custom control to provide extended functionality.

Now you get to set the default properties of each device.  If you wish the property to be saved and used at startup, you much check the box to the left indicating that this is a save property.

 

Now you will get a confused screen with all the choices that you have made organized onto the screen.

First you need to set up the screen as you would like to work.  Start by putting the Joystick where it is comfortable, then it is probably desired to have the Image Viewer on top (tabs in the upper left of the screen). 

            Now you should setup the device adapter preferences.  I generally use the Record and Save Tools.  This tab is in the lower right of the screen.  You can choose to manipulate any device’s properties from these tabs. 

            Once the screen looks nice, you should give the configuration a name and then click Save Profile.  It should look something like this

 

You may have to restart the program at this point due to the behavior of some of the device adapters, but try to get started first.  In the Record and Save Tools, you can start the camera in focus mode (i.e. just showing images, no saving) by clicking the start button in the Focus Mode group.  The images will stream to the screen.  In order to record images, use the Record mode group.  The images will be saved to the experiment folder.

 

            To change the properties of the camera, change to the Camera Properties tab in the lower right of the screen, and then you will gain access to the camera.

This tab lets you set some of the common camera properties and functions.  To reach even more of the Camera’s properties, click the Advanced tab in the upper right of the screen. Here you will find all the properties of the camera and can adjust them as needed.  This pattern is the same for all the device adapters.

You should be able to set up the microscope and image at this point.
