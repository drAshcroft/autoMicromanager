Installation

      Scilab provides a program called DotNet-Component-Scilab.dll to provide communication between a .Net program and Scilab.  The installation program the file for Scilab 5.2.1, but if you have another version, you will have to compile this program for yourself.

 

You must navigate to

 

C:\Program Files\scilab-x.x.x\modules\call_scilab\examples\call_scilab\NET\C#\DotNetScilab.sln. 

 

Where scilab-x.x.x must be changed for your version.

 

You must then compile this program.  The .dll will be compiled and then installed into the correct bin directory.

 

 

Usage

 

      Scilab only allows the .Net application to call into the library.  In order to use this, you must use the scripting control in the stand alone application.  Set up your hardware as desired with the hardware wizard and then switch to the IronPython ScriptControl.  At the top of the page is a menu item “Script Templates”  included in this template is a SciLab template that provides an example of how to load data into the Scilab control, manipulate it and then have it sent back to the standalone application.  An example script is included here.  The language is IronPython, which is the .Net adaptation of python.  Python is a relatively simple scripting language that provides all the power needed for imaging applications.

 

import clr

clr.AddReference('System.Windows.Forms')

from System.Windows.Forms import *

clr.AddReferenceToFileAndPath('c:\\program files\\scilab-5.2.1\\bin\\dotnet-component-scilab.dll')

import DotNetScilab

from DotNetScilab import *

 

#Create a scilab connection

oScilab = Scilab(True)

 

#Indicate that you wish to receive images from the camera

Console.RegisterAsImageProcessor('IgorProcess')

#Procedure to process the images

def ImageCatch(CoreImages):

    Console.WriteLine('Number Of Channels:' + len(CoreImages).ToString())

    #images are always returned as an array that contains the stack/montage/channels

    Data= CoreImages[0].GetLinearArrayDouble()

    mA = CoreImages[0].Width

    nA = CoreImages[0].Height

    #send the data to Scilab

    oScilab.createNamedMatrixOfDouble('ImageMatrix', mA, nA, Data)

    ret =oScilab.SendScilabJob('d = 0:' + (mA-1).ToString() + ';')

    Console.WriteLine(ret)

    ret=oScilab.SendScilabJob('plot3d(d,d,ImageMatrix);')

    Console.WriteLine(ret)

    ret=oScilab.SendScilabJob('ImageMatrix = ImageMatrix .* ImageMatrix')

    Console.WriteLine(ret)

    ret = oScilab.SendScilabJob('m=min(ImageMatrix)')

    m=oScilab.readNamedMatrixOfDouble('m')

    Console.WriteLine('Min Value is :' + m[0].ToString() )

    #get processed image back from Scilab

    DimB = oScilab.getNamedVarDimension('ImageMatrix')

    Data= oScilab.readNamedMatrixOfDouble('ImageMatrix')

    #place the image back in the images array

    CoreImages[0]= CoreImages[0].CreateImageFromArray(Data,DimB[0],DimB[1])

    #you must return an image array of the same dimension as was aquired

    return CoreImages

 

#Allows scripting to finish

def EndImageCatch():

    Console.WriteLine('Stopping Eventhandlers')

    Console.KillRequested -= EndImageCatch;

    Console.RemoveImageProcessor('IgorProcess')

    Console.ImageProduced -= ImageCatch

 

Console.ImageProduced += ImageCatch

Console.KillRequested += EndImageCatch

 

#Get camera device and change the exposure.

camera=ECore.GetDevice('camera')

camera.SetDeviceProperty('Exposure','100')

Console.WriteLine(camera.GetType().ToString())
